▼変更されたＳＰＥ

　「蟹」において効果や取得条件の変更を受けたＳＰＥたち。
　変更された点はオレンジ色で表記している。

絆

■効果 /

セッションスタート時、因子ダイス＋３。

又、自身が使用する、
あるいはされるＡＥ【エール】の補正をさらに＋１Ｄ６。

■使用制限 / なし
■消費 / なし

限界突破
■効果 /

ダメージロールの判定結果を２倍にする。
■使用制限 / セッション中１回
■コスト /

自壊ダメージ
　＋　
次ターンポストアクションまでＡＥ宣言不可。

又、同様に次ターンポストアクションまで
自身の能力によるプラス補正効果が得られない。
味方からの補正やアイテムによる補正は可能。

■補足 /

命中判定前に事前宣言が必要。

ＳＥ【オールレンジ】【パーフェクト】
特殊武装【ロケットランチャー】【手榴弾】
消費武装【使い捨てランチャー】【違法グレネード】
 【ダメージバースト】と併用不可。

パージ
■効果 /

特殊武装によるスペック圧迫がゼロになる。
パージ後は無効化された圧迫スペックの値だけ
行動値に常時ボーナスを得る。
■使用制限 / セッション中１回
■消費 / なし
■補足 /

使用後は特殊武装が使用不可となる。
アタッチメントはこの限りではない。
カルマ【武器屋】による特殊武装も同様に使用不可能となるが、
カルマによる武装装備はスペックの圧迫を行っていないため
行動値へのボーナスは得られない。

パージ後にクリーチャーを創造する場合、
発動スペックはパージ後のものを参照する。

奥義
■効果 /

通常攻撃に防御無視効果を付与。
基本的に常時適用。ただし任意でＯＦＦも可能。

■使用制限 / なし
■消費 / なし
■補足 / ＳＥ【リフレクション】と貫通対抗を行う場合、
自動成功扱いとしてそのままダメージロールを行う。

▼追加されたＳＰＥ

　「蟹」で新たに追加されたＳＰＥ。
　構築の幅を広げる上で一助となれば幸いである。

切札
■効果 /

そのアクション中のみ、
能力精度が倍、ＳＯ無視、ＰＰ消費半減の効果を得る。

■使用制限 / シナリオ中１回のみ
■消費 / なし
■補足 / 暴走率１２０％以上でなければ宣言不可。
